Get up and running with Briefly CRM in just a few minutes.

<Callout title="Prerequisites">
Before you begin, make sure you have **Bun** (v1.0.0+), **PostgreSQL**, and **Redis** installed on your system.
</Callout>

## Quick Start

Follow these steps to get your CRM running:

| Step | Description | Time |
| ---- | ----------- | ----- |
| 1 | Clone the repository | 1 min |
| 2 | Install dependencies | 2 min |
| 3 | Configure environment | 5 min |
| 4 | Initialize database | 3 min |
| 5 | Start the server | 1 min |
