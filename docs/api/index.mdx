This API provides comprehensive endpoints for managing your E-Commerce CRM, including customers, products, orders, segments, and integrations.

## Authentication

All API endpoints require Bearer token authentication. Include your API key in the Authorization header:

```bash
Authorization: Bearer your_api_key
```

## Base URL

```
/api
```
